# TaskHub
TaskHub é um sistema colaborativo de gestão de tarefas desenvolvido com foco em arquitetura distribuída, boas práticas backend, mensageria assíncrona, segurança, documentação clara e notificações em tempo real. O projeto demonstra habilidades práticas em ASP.NET Core, RabbitMQ, SignalR, JWT, EF Core, Docker e organização de serviços independentes.
